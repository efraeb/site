# Sample Project

A demonstration of the markdown-based portfolio system.

## Overview

This is a sample project to show how the portfolio works. You can write your project descriptions in markdown and they will be rendered with a clean, minimalist design.

## Features

- **Simple markdown editing** - Just write markdown files
- **Clean design** - Inspired by Claude Code's aesthetic
- **Easy to maintain** - No complex build process

## Technical Details

The portfolio uses:

- Pure HTML/CSS/JavaScript
- Marked.js for markdown parsing
- Monospace typography
- Dark theme with minimal colors

function greet(name) {
